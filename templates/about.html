About Hy

The Hy programming language is an open source lisp dialect which converts directly to python's abstract syntaxt tree. This allows for easy, complete interoperability between these languages. Hy compiles to python bytecode, which can be run on a python interpreter, and it can load arbitrary python modules directly. That means that the entire python standard library is directly available in Hy! This site provides an editor and interpreter which may be used to try out the language.
